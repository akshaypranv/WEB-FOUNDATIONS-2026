# 🌐 Web Foundations 2026

This repository is a structured log of my journey into **Full-Stack Engineering**. It documents my mastery of the core building blocks of the web, moving from semantic structure to advanced styling and layout logic.

## 🏗️ Learning Modules & Projects

### 1. HTML Architecture & Semantics
Focused on building clean, SEO-friendly, and accessible document structures.
* **Movie Ranking Project:** Practice with ordered/unordered lists and semantic nesting.
* **Birthday Invite:** Implementing media assets (images) and text hierarchy.
* **Demo Portfolio:** A structural blueprint for a multi-section personal site.

### 2. CSS Engineering & Styling
Moving beyond basic colors to understand how the browser renders and positions elements.
* **The Box Model (Module 6.3):** Deep dive into `padding`, `border`, `margin`, and `content` calculations. Mastered the difference between `border-box` and `content-box`.
* **Positioning Logic:** Practical application of `static`, `relative`, `absolute`, and `fixed` positioning to control UI flow.
* **Selector Combinations:** Advanced targeting of elements using descendant, child, and adjacent sibling selectors to keep CSS "DRY" (Don't Repeat Yourself).

### 3. Layout Engineering & Responsiveness
Moving from static documents to dynamic, screen-aware interfaces.
* **Legacy Layouts (Floats):** Understanding how to clear floats and use them for wrapping content—a necessary skill for maintaining older codebases.
* **Flexbox Mastery:** Implementing 1D layouts using `justify-content`, `align-items`, and `flex-direction` to build modern, fluid UI components.
* **Media Queries:** The core of Responsive Design. Utilizing breakpoints to ensure layouts adapt seamlessly from mobile devices to desktop monitors.
